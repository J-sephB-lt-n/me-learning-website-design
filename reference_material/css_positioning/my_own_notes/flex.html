Items within a flex-container can be ordered using the 'order' attribute:

#num1 {
 order: 2;
}
#num2 {
 order: 3;
}
#num3 {
 order: 1;

'flex-basis' sets the initial width of an item (it still grows and shrinks after this):

#num1 {
 flex-basis: 80px;
}

1
2
3

the 'flex' property can be used as shorthand for 'flex-grow', 'flex-shrink' and 'flex-basis'

e.g.
x {
 flex: 0 0 200px;
}
is the same as
x {
 flex-grow: 0;
 flex-shrink: 0;
 flex-basis: 200px;
}

'align-self' sets the 'align-items' attribute for a single item

#num1 {
 align-self: center;
}
#num2 {
 align-self: flex-start;
}
#num3 {
 align-self: flex-end;
}
